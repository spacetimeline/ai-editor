# 字体配置

## 示例代码

```typescript
new AiEditor({
    element: "#aiEditor",
    fontFamily:{
        values:[
            {name: "宋体", value: "SimSun"},
            {name: "仿宋", value: "FangSong"},
            {name: "黑体", value: "SimHei"},
            {name: "楷体", value: "KaiTi"},
            {name: "微软雅黑", value: "Microsoft YaHei"},
            {name: "方正仿宋简体_GBK", value: "FangSong_GB2312"},
            {name: "Arial", value: "Arial"},
        ]
    },
})
```
以上代码中，name 表示字体名称，value 表示 css 使用的字体样式。
