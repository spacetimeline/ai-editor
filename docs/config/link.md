# Hyperlink configuration


## Samples

```typescript
new AiEditor({
    element: "#aiEditor",
    link: {
        autolink: true,
        rel: "",
        class: "",
    }
})
```

- **autolink**: Auto-connect
- **rel**: The default value configuration for the `ref` attribute of the `a` tag.
- **class**:  The default value configuration for the `class` attribute of the `a` tag.
